First-time setup (run this SQL once)

Supabase dashboard → SQL Editor → paste & Run:

create table if not exists stock(
  id text primary key, ref text, name text, size text, color text,
  available int default 0, exp_soon int default 0, hold int default 0,
  damaged int default 0, thresh int default 5,
  updated_at timestamptz default now());
alter table stock enable row level security;
create policy "team access" on stock
  for all using (true) with check (true);

Then Project Settings → API → copy the Project URL and the anon public key into the boxes above.
